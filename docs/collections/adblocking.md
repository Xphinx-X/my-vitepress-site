# 🛡️ Adblocking

Block ads, pop-ups, and trackers everywhere.

::: warning Important
Never browse the modern web without an adblocker. It's not just about ads; it's about security and privacy!
:::

## 🧰 Extensions
- **[uBlock Origin](https://ublockorigin.com/)** <Badge type="tip" text="Recommended" /> - The gold standard for adblocking.
- **[AdGuard](https://adguard.com/)** - Great premium alternative with extra features.
- **[Privacy Badger](https://privacybadger.org/)** - Automatically learns to block invisible trackers.

## 📱 DNS Adblocking
- **[NextDNS](https://nextdns.io/)** - Block ads at the network level for your whole house.
- **[AdGuard DNS](https://adguard-dns.io/)** - Easy to set up private DNS.
