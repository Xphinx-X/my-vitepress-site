# 🛠️ Cool Tools

A handpicked list of software and apps that make life easier.

::: tip Recommendation
Always prioritize **Open-Source** software when possible to protect your privacy and support the community!
:::

## 🎨 Design
- **[Figma](https://figma.com)** <Badge type="info" text="Freemium" /> - Collaborative design tool for building meaningful products.
- **[Penpot](https://penpot.app)** <Badge type="tip" text="Open Source" /> - Open-source design and prototyping platform.

## 💻 Development
- **[VSCode](https://code.visualstudio.com)** - The code editor where the magic happens.
- **[VitePress](https://vitepress.dev)** <Badge type="tip" text="Open Source" /> - The tool that built this very site!
- **[GitHub](https://github.com)** - Where the world builds software.

- **[Notion](https://notion.so)** <Badge type="info" text="Freemium" /> - All-in-one workspace for notes, tasks, and wikis.

- **[Obsidian](https://obsidian.md)** <Badge type="tip" text="Local First" /> - A second brain, for you forever.
