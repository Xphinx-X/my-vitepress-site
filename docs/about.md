# ℹ️ About Xphinx Collection

Welcome to the Xphinx Collection! This is a curated wiki dedicated to gathering fascinating, useful, and open-source tools from across the internet.

::: tip Why does this exist?
The internet is vast, and finding reliable, safe, and ad-free tools can be hard. This collection filters out the noise and provides direct, community-vetted recommendations.
:::

## 🌍 How to Navigate
- **Search Bar:** Use the search bar at the top to instantly find specific tools.
- **Sidebar:** Use the menu on the left to browse through categories.
- **Grid Cards:** Click the cards on the homepage to dive into a topic.

## 🤝 Contributing
If you find something fascinating that isn't on this list, feel free to hit the **"Edit this page on GitHub"** link at the bottom of any page to add it!
